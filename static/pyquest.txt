Что такое класс в Python?
Класс — это шаблон для создания объектов
Класс — это функция, которая возвращает объект
Класс — это модуль в Python
Класс — это тип данных, который нельзя изменять

Что такое объект в Python?
Объект — это экземпляр класса
Объект — это функция, которая вызывается внутри класса
Объект — это модуль, который содержит классы
Объект — это тип данных, который нельзя изменять

Что такое наследование в Python?
Механизм, позволяющий создавать новый класс на основе существующего
Механизм, который позволяет копировать код из одного класса в другой
Механизм, который запрещает использование методов родительского класса
Механизм, который удаляет методы родительского класса

Что такое инкапсуляция в Python?
Принцип, который объединяет данные и методы, работающие с ними, в одном классе
Принцип, который разделяет данные и методы на разные модули
Принцип, который запрещает доступ к данным класса
Принцип, который удаляет данные из класса

Что такое полиморфизм в Python?
Возможность использовать одинаковые имена для методов, которые работают по-разному в разных классах
Возможность создавать множество классов с одинаковыми именами
Возможность изменять тип данных объекта во время выполнения программы
Возможность создавать объекты без использования классов

Что такое метод в Python?
Функция, определенная внутри класса
Функция, которая вызывается только вне класса
Функция, которая не может принимать аргументы
Функция, которая автоматически вызывается при создании объекта

Что такое self в Python?
Ссылка на текущий экземпляр класса
Ссылка на родительский класс
Ссылка на глобальную переменную
Ссылка на модуль, в котором определен класс

Что такое конструктор в Python?
Метод __init__, который вызывается при создании объекта
Метод __del__, который вызывается при удалении объекта
Метод __call__, который вызывается при вызове объекта как функции
Метод __str__, который вызывается при преобразовании объекта в строку

Что такое статический метод в Python?
Метод, который принадлежит классу, а не экземпляру класса
Метод, который нельзя вызывать из экземпляра класса
Метод, который автоматически вызывается при создании объекта
Метод, который изменяет состояние класса

Что такое декоратор @classmethod в Python?
Метод, который принимает класс в качестве первого аргумента
Метод, который принимает экземпляр класса в качестве первого аргумента
Метод, который нельзя вызывать из экземпляра класса
Метод, который автоматически вызывается при удалении объекта
